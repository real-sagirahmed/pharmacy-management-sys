# Manual Guide: Branding Changes (Title & Favicon)

এই গাইডটি আপনাকে দেখাবে কিভাবে ম্যানুয়ালি আপনার প্রজেক্টের টাইটেল এবং ফেভিকন (Favicon) পরিবর্তন করতে হয়।

### ১. টাইটেল পরিবর্তন (Title Change)
ব্রাউজারের ট্যাবে যে নাম দেখা যায় সেটি পরিবর্তন করতে:
- `frontend/src/index.html` ফাইলটি ওপেন করুন।
- `<title>...</title>` ট্যাগের মাঝখানের লেখাটি আপনার পছন্দমতো পরিবর্তন করুন।
  *উদাহরণ:* `<title>s7- Drug House | PharmacySys</title>`

### ২. ফেভিকন পরিবর্তন (Favicon Change)
ফেভিকন হলো ব্রাউজার ট্যাবে নামের পাশে থাকা ছোট আইকন।
- আপনার নতুন আইকনটি (PNG বা ICO ফরম্যাটে) `frontend/public/` ফোল্ডারে রাখুন।
- ফাইলটির নাম দিন `favicon.png` (বা আপনার পছন্দের নাম)।
- এরপর `frontend/src/index.html` ফাইলে নিচের লাইনটি আপডেট করুন:
  `<link rel="icon" type="image/png" href="favicon.png">`

### ৩. কনফিগারেশন চেক (Angular Configuration)
অ্যাঙ্গুলার যাতে আপনার আইকনটি বিল্ড করার সময় খুঁজে পায়, সেজন্য:
- `angular.json` ফাইলে `assets` সেকশনে গিয়ে দেখুন `public` ফোল্ডারটি ইনক্লুড করা আছে কিনা।
- সাধারণত নিচের মতো থাকে:
  ```json
  "assets": [
    { "glob": "**/*", "input": "public" }
  ]
  ```

### ৪. লাইভ সাইটে আপডেট করা (Update Live Site)
আপনার প্রজেক্টটি **GitHub Actions**-এর মাধ্যমে অটোমেটিকভাবে ডেপলয় হয়। তাই লোকাল পরিবর্তনগুলো লাইভ সাইটে দেখতে হলে:
- আপনার সব পরিবর্তন GitHub-এ `push` করতে হবে।
- কমান্ড:
  ```bash
  git add .
  git commit -m "Branding update"
  git push origin main
  ```
- পুশ করার পর ২-৩ মিনিট অপেক্ষা করুন, GitHub Actions বিল্ড শেষ হলে সাইট অটোমেটিক আপডেট হয়ে যাবে।

### ৫. ক্লিন ডেপলয়মেন্ট টিপস (Clean Deployment Tips)
আপনার প্রজেক্টে এখন **অটোমেটেড ডেপলয়মেন্ট স্ট্যাবিলিটি (Phase 9)** যুক্ত করা হয়েছে। এর ফলে:
- ডেপলয়মেন্ট শুরু হলে সাইট অটোমেটিক আপডেট মোডে চলে যাবে।
- আপডেট শেষ হলে সিস্টেম নিজে থেকেই সাইটকে অনলাইনে ফিরিয়ে আনবে।
- যদি কখনো বড় কোনো সমস্যার কারণে সাইট "System Update in Progress" এ আটকে থাকে, তবে ম্যানুয়ালি `wwwroot` ফোল্ডার থেকে `app_offline.htm` ফাইলটি ডিলিট করে দিলেই সব ঠিক হয়ে যাবে।

### ৬. নতুন UI/UX ফিচার এবং ব্যবহার নির্দেশিকা (UI/UX Guide)
সম্পূর্ণ প্রজেক্টের UI রিডিজাইন করার ফলে কিছু নতুন এবং আধুনিক ফিচার যুক্ত হয়েছে:

- **Dashboard:** এখন ড্যাশবোর্ডেই ওষুধের সংখ্যা, লো-স্টক অ্যালার্ট এবং বিক্রির হিসাব দেখা যাবে। মোবাইল থেকে দেখলে বাম পাশের সাইডবারটি ১টি বাটন দিয়ে হাইড/শো করা যাবে।
- **Medicine Search:** ওষুধ খুঁজতে এখন আর পুরো টেবিল স্ক্রল করতে হবে না। `Medicines` পেজে একটি সার্চ বার আছে যেখানে নাম বা জেনেরিক নাম লিখলে সাথে সাথে ডেটা ফিল্টার হবে।
- **Sales POS (পয়েন্ট অফ সেল):** এখানে পেমেন্ট করার জন্য Cash, Card বা Mobile বাটন সিলেক্ট করা যাবে। গ্র্যান্ড টোটাল এখন রিয়েল-টাইমে আপডেট হয়।
- **User Management:** সকল ইউজারের নামের প্রথম অক্ষর দিয়ে রঙিন অটো-জেনারেটেড অভাতার (Avatars) তৈরি করা হয়েছে। অ্যাডমিন প্যানেল থেকে ইউজারের রোল পরিবর্তন করলে তা সাথে সাথে সেভ হবে।
- **Auth Pages:** লগইন এবং রেজিস্ট্রেশন পেজগুলো এখন অনেকটা আধুনিক অ্যাপসের মতো (Split Layout)। বাম পাশে ব্রান্ডিং এবং ডান পাশে ফর্ম থাকবে।

### ৭. লোকাল ডেভেলপমেন্ট চেক (Local Check)
যেকোনো বড় পরিবর্তনের পর লোকাল-এ চেক করতে:
1. `frontend` ফোল্ডারে টার্মিনাল ওপেন করুন।
2. `npm run build` দিয়ে দেখুন কোনো এরর আসে কিনা।
3. `npm run start` দিয়ে ব্রাউজারে `localhost:4200`-এ চেক করুন।

- **Tailwind CSS & CSS Variables:** কাস্টম থিমিং এবং রেসপনসিভ ডিজাইন।
- **GitHub Actions:** অটোমেটেড ডেপলয়মেন্টের জন্য।
